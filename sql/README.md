# Scripts SQL

Scripts para manutenção e correção do banco de dados.

## Arquivos

- **`fix_columns.sql`**: Script original para corrigir colunas do banco de dados

- **`fix_columns_corrected.sql`**: Versão corrigida do script de correção de colunas

## Como usar

Execute os scripts SQL diretamente no PostgreSQL:

```sql
-- Via psql
\i fix_columns_corrected.sql

-- Ou através de um aplicativo de banco de dados
```
