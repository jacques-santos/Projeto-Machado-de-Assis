Créditos

Sobre este projeto

Banco de Dados Machado de Assis é um catálogo digital dedicado às obras, registros e documentação relacionados a Joaquim Maria Machado de Assis, um dos maiores escritores da literatura brasileira.

Dados

Os dados preservados neste catálogo foram compilados com dedicação ao estudo da obra machadiana. Este banco de dados funciona como uma ferramenta de pesquisa para estudiosos, pesquisadores e leitores interessados em explorar a produção literária de Machado de Assis de forma sistêmica.

Tecnologia

Este projeto foi desenvolvido com tecnologias open-source:

  • Backend: Django e Django REST Framework
  • Frontend: HTML, CSS e JavaScript puro
  • Banco de dados: PostgreSQL

Licença e Uso

Este catálogo está disponível como recurso educacional e de pesquisa. As obras de Machado de Assis estão em domínio público.
